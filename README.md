# Pre Requisites
- rust
- nginx
- node
- docker

# Local Installation
``` git clone https://github.com/Arpit078/burner-browsers.git ```

``` cd burner-browsers ```

``` cd backend ```

``` cargo build ```

``` sudo target/debug/burner-browsers ```

``` cd .. ```

``` cd frontend ```

``` npm install ```

``` npm start ```
